## Table Policy

- Printed page numbers are Andrews book page numbers. `ANDREWS_TRANSCRIPTION_CANVAS.md` headings use PDF page numbers.
- Keep table cells as cells. Blank cells are meaningful: they show that a stem shape is not used in that tense/mood environment.

- Appendix tables are validation/paradigm witnesses. They do not license new derivational routes by themselves; the relevant lesson rules must still authorize the route.
